接上 AI，只需 3 步

用你自己的 API key——额度花在自己账上，数据只走你和 AI 平台之间。整个过程像注册个网盘：注册 → 复制 key → 粘贴，5 分钟只弄一次。

1选一家 AI 平台

DeepSeek：手机号就能注册，新用户送免费额度；之后一次拆解也就几分钱。

2把 key 粘贴到这里

注册后进「API Keys」页面 → 点创建 → 复制那串 sk- 开头的字符，回来粘贴。

3连接，挑个「会思考」的模型
